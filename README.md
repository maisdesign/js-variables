# JS Variables

Esercizi base su variabili, numeri e stringhe in JavaScript. Il progetto raccoglie tre mini‑esercizi con `index.html` e `script.js` separati per argomento.

## Struttura del progetto
- `01-numbers/` esercizi con operatori numerici e operazioni aritmetiche
- `02-strings/` esercizi su variabili e proprietà delle stringhe
- `03-strings-bonus/` esercizi bonus su metodi delle stringhe

## Come usare
1. Apri una delle cartelle (es. `01-numbers/`).
2. Apri `index.html` nel browser.
3. Apri la console del browser per vedere l’output dei `console.log`.

## Contenuto degli esercizi
- **Numeri:** somma, prodotto, divisione, sottrazione, somme combinate.
- **Stringhe:** dichiarazioni, concatenazione, lunghezza, metodi `indexOf`, `substring`, `replace`, template literal, `toUpperCase`.

## Note
- Ogni cartella è indipendente.
- I file `script.js` sono già completati con gli output attesi nei commenti.
